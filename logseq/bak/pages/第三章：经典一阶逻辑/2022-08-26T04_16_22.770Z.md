- 并不是所有的推理都能通过经典命题逻辑表达
- 比如一个命题给出某个集合中的所有元素都拥有某种性质，而另一个命题给出某个元素属于该集合，则能够推出该元素拥有该性质
- 对于这样的逻辑，需要引入量词逻辑来描述
- # 量词
	- **结构（structure）**，包含：
		- **论域（domain）** ：所研究对象构成的不空集，即研究的范围
		- **个体（individual）**：所研究的对象
		- **关系（relation）**：个体之间构成的关系
		- **函数（function）**：作用在个体上的函数
	- 例如自然数是一个结构，包含：
		- 论域：自然数集合N
		- 个体：0
		- 关系：相等
		- 三个函数：
			- 后继（'）
			- +
			- *
	- 在描述很多问题是，需要引入**变元**来增加表达能力
	- **量词**包含两个连接词
		- 全称量词（universal quantifier）：对于所有论域中的个体
		- 存在量词（existential quantifier）：存在论域中的个体
		- 使用量词构造全称命题和存在命题是简单命题
	- **命题函数（proposition function）**
		- 在论域N上有定义
		- 把某一个体指派为其接收变元的值时，它构成一个命题（输入 一个个体，输出一个命题）
		- 论域D上的n元命题函数是把$D^n$映射到{1,0}中的n元函数
	- **自由变元（free variable）**与**约束变元（bound variable）**
		- 一个变元本身的作用是作为一个可以被赋值的变量
		- 被量词规定范围的变元不再是一个可以被传入任何定义域内值的变量，而是指代全体拥有某一性质的个体，因此其被称作约束变元，也称被**量化的变元**
		- 没有被量化的变元被称为自由变元
	- 本节讨论的所有量词和变元都是针对个体的，而针对论域的子集和笛卡尔积的变元和量词是二阶逻辑的内容
- # 一阶语言
	- 用符号$\mathscr{L}$表示
	- ## 符号
		- **个体符号**： 小写字母a,b,c
		- **关系符号**：大写字母F,G,H
			- 每一个关系符号都有确定的元数$n\ge 1$
			- **相等符号（equality symbol）**：一种特殊的符号，记作$\equiv$
				- 任何其他关系符号都不是相等符号
		- **函数符号**：小写字母f,g,h
			- 任何函数符号有自己确定的元数
		- **自由变元符号**和**约束变元符号**
			- 自由变元：小写字母：u,v,w
			- 约束变元：小写字母：x,y,z
		- **联结符号**
			- $\neg\qquad \wedge\qquad\vee\qquad\rightarrow\qquad\leftrightarrow$
		- **量词符号**
			- $\forall \qquad \exists$
		- **标点符号（punctuation）**
			- (          )
	- ## 项（Term）
		- **定义3.2.1** $t\in Term(\mathscr{L})$，当且仅当其能由有限次使用以下法则生成
			- i) $a,u,\in Term(\mathscr{L})$，即单独一个个体符号和自由变元符号是项
			- ii) 如果$t_1,\cdots,t_n \in Term(\mathscr{L})$，并且f是n元函数符号，则$f(t_1,\cdots,t_n)\in Term(\mathscr{L})$
		- 不含自由变元符号的项是**闭项**
	- ## 原子公式（Atom）
		- $\mathscr{L}$的表达式是Atom($\mathscr{L}$)的元，当且仅当它有以下两种形式之一
			- i) $F(t_1,\cdots,t_n)$，其中的F是n元关系符号，并且$t_1,\cdots,t_n \in Term(\mathscr{L})$
			- ii) $\equiv(t_1,t_2),\qquad t_1,t_2\in Term(\mathscr{L})$
				- 可简写为$t_1\equiv t_2$
			- 直观地说，就是里面是项，外面是关系符号
	- 令$U,V_1,\cdots,V_n$是$\mathscr{L}$的表达式，$s_1,\cdots,s_n$是$\mathscr{L}$的符号。把U写成：$U(s_1,\cdots,s_n)$，表示$s_1,\cdots,s_n$在U中出现
	- 随后出现的$U(V_1,\cdots V_n)$是把$V_i$代入U中$s_i$后得到的表达式
	- ## 公式
		- 见定义3.2.3（p87）
		- **拟公式**：有量词约束但是去掉了量词的公式
- # 语义
  collapsed:: true
	- DONE 把这一节看完
	- 如果是一个针对具体结构的一阶语言，个体符号和个体，函数符号和函数，关系符号和关系，都是一一对应的关系，这三个集合往往是有限的
	- 对于抽象的，一般的一阶语言，仍然需要定义相关的东西：
		- 论域：一个不空集
		- 个体符号：表示论域中的个体，可数无限多
			- 不同的个体符号可以解释为论域中的同一个个体，也可以解释为不同的个体
		- 关系符号：表示论域中个体的关系，可数无限多
		- 函数符号：表示可作用于论域中个体的函数，可数无限多
			- 表示的函数都是论域上的全函数（处处有定义）
		- 八类符号都是表示论域中的个体（标点符号除外）
	- 原子公式表示论域中的n个个体有R关系，是一个命题
	- **一个闭项表示一个个体**，可能是原子个体，也有可能通过函数映射的得到个体
		- 一个含有自由变元符号的项，经过解释，表示一个在论域上的函数，**在对变元指派论域中的个体**之后，成为一个个体
	- 同理，**一个闭公式表示一个命题**，一个含有变元的原子公式表示一个命题函数，在经过指派之后成为一个命题
	- 个体符号和自由变元符号最终都会表示某一个个体，**但它们有区别**
		- 个体符号本来就是表示个体的，经过解释即可表示个体
		- 自由变元符号经过解释之后成为论域上的变量，需要指派个体之后才能表示个体
	- 一次指派同时给所有可数无限多的自由变元做指派
		- 但是实际操作中，一般只考虑出现在研究公式中的自由变元的指派，这些变元虽然多但往往是有限的
		- 对于没有出现在目前考虑的的公式中的自由变元，其指派可以被忽略
	- 同一类的符号（个体符号，函数符号，关系符号），经过解释之后**可以表示相同的**（个体，函数，关系），**也可以表示不同的**（个体，函数，符号）
		- 就好比两个自然数变量，x，y，它们都可以表示5，也可以一个表示4，一个表示6
	- ## 赋值（valuation）
		- 之前的文本中出现了多次“解释”和“指派”，但是还没有对其做定义
			- 解释作用于：
				- 个体符号，使其成为论域中的个体
				- 函数符号，使其成为论域上的函数
				- 关系符号，使其成为论域中个体之间的关系
			- 指派指作用于自由变元
		- 我们将解释和指派的过程合起来成为“**赋值**”
		- 赋值以解释的论域作为其论域
		- ### 关系
			- S上的n元关系的外延是：
				- $\{<a_1,\cdots,a_n> | a_1,\cdots,a_n\in S, 并且它们有R关系\}$
			- S上的相等关系的外延是：
				- $\{<x,y>| x,y\in S, 并且x=y\}\cup \{<x,x>| x\in S\}$
		- ### 赋值的定义 （定义3.3.1）
		  collapsed:: true
			- 对于一阶语言$\mathscr{L}$的**赋值**$v$包括：
				- 一个论域$D$
				- 一个函数，记作$v$
			- 其定义在四种符号上：
				- 个体符号：$v$(a)
				- 关系符号：$v(F)$，$v(\equiv)$
				- 函数符号：$v$(f)
				- 自由变元符号：$v$(u)
				- 分别可写作：$\mathrm{a}^v,\mathrm{F}^v,\equiv^v,\mathrm{f}^v,\mathrm{u}^v$
			- **结果**
				- 对于个体符号和自由变元符号：
					- $\mathrm{a}^v,\mathrm{u}^v\in D$
					- 虽然结果一样，都是个体，但是对于变元符号的赋值包括两步
				- 对于关系符号：
					- $\mathrm{F}^v \subseteq D^n$
					- $\equiv^v = \{<x,x> | x\in D\} \subseteq D^2$
				- 对于函数符号
					- $\mathrm{f}^v : D^m\rightarrow D$
		- ### 项的值（定义3.3.2）
			- $\mathscr{L}$的项在以$D$为论域的赋值$v$下的**值**递归定义如下：
				- 对于个体符号，自由变元符号：$\mathrm{a}^v,\mathrm{u}^v\in D$
				- 对于其它项：$f(t_1,\cdots,t_n)^v = f^v(t_1^v,\cdots,t_n^v)$
		- ### 赋值的变换
			- 假设一个论域D，一个赋值v，一个自由变元u ，一个个体a：
				- v(u/a)是一个新的以D为论域的赋值
				- $u^{v(u/a)}=a$
				- 除此之外和v没有区别
		- ### 公式的（真假）值（定义3.3.4）
			- 见书上p98
			- 一个公式是**可满足的**，当且仅当有一个（以某个不空集为论域的）赋值使其值为1
			- 一个公式是**有效**的，当且仅当任何（以某个不空集为论域的）赋值都使其值为1（类似于命题逻辑中的重言式）
			- 在无限集论域上做有限性判断是不可行的
				- 没法给出一个统一的算法判断公式的有效性
				- 只能针对单个算法给出专门的判断算法

- # 逻辑推演
	- 各种定理见教材p109
	- 对于$\exists +$的使用，往往通过构造性证明或$\neg -$进行
		- 所谓构造性证明往往是真的找出论域中的一个个体来证明论域中的某一个个体拥有该性质
		- 也叫构造出一个个体
	- 同一个约束变元符号不能在另一个量词的辖域中出现
		- 例如：$\forall x \forall x F(x,x)$是不正确的，其不能成为一个原子公式
	- 引入两个新的量词：
		- $\exists !!xA(x)$表示$\forall x\forall y [A(x)\wedge A(y)\rightarrow x\equiv y]$
			- 存在至多一个个体满足A性质
		- $\exists !xA(x)$表示$\exists x[A(x)\wedge\forall y(A(y)\rightarrow x\equiv y)]$
			- 存在恰好一个个体满足A性质
		- 朴素的$\exists$表示至少存在一个
- # 前束范式（prenex normal form）
	- 一阶逻辑的一种范式
	- $Q_1x_1,\cdots,Q_nx_nB$
	- $Q_ix_i$为**前束词**（prefix）
	- $B$称为**母词**（matrix）
	- 任何一个一阶命题逻辑公式都可以变成一个前束范式，且该前束范式与该公式互相可推导，互相可推演
